## Laravel Project "Movies"  

Quick Installation:

 - git clone https://github.com/nidhal-barkit/Movies.git.
  
 - cd movies
  
 - composer install
 
 - cp .env.example .env
 
 - create database and set the dbname in .env
 
  
 - php artisan migrate
 
 - php artisan db:seed
 
 - Add Role 'User' in table roles
 
 - php artisan serve.

=> A User/Admin can login if his status is active
